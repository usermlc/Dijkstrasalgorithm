Алгоритм Дейкстри

На цій сторінці реалізовано алгоритм Дейкстри для знаходження найкоротших шляхів від однієї вершини до всіх інших у зваженому графі без від'ємних ваг.

Використані структури даних:

  • Граф: представлений як об'єкт об'єктів для зберігання зв'язків між вершинами та вагами ребер.
  • Черга з пріоритетом: для ефективного вибору наступної вершини для обробки.

Реалізація алгоритму: Алгоритм проходить через граф, використовуючи чергу з пріоритетом, щоб знайти найкоротший шлях від стартової вершини до кожної іншої вершини, оновлюючи відстані та попередників по мірі проходження.
